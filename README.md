Pepper
======

What if programming could turn into something we can all understand? Enter Pepper, the best programming language ever.

Check out getting_started.pepper to start using Pepper now.

<strong>Installation</strong>

Install Python 3.0 on to your computer. This is required to use Pepper. Check out <a href="python.org">python.org</a> for more details. Python 3.0 is preinstalled on Mac OS and most Linux systems.

Then download pepper and place it on your Desktop.

<strong>Running Pepper Programs</strong>

On Windows:
<br/>
<code>
cd /Users/<em>Your Username</em>/Desktop</code>
<br/>
<code>
pepper <em>path to pepper program</em>
</code>

On Mac:
<br/>
<code>
cd ~/Desktop
</code>
<br/>
<code>
pepper <em>path to pepper program</em>
</code>
